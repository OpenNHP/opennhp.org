Notre mission

La technologie au service de l'humanité À but non lucratif

Internet est le foyer numérique partagé de l'humanité. L'IA est une arme à double tranchant — elle peut nous rendre plus forts ou nous mettre en danger. Nous construisons une communauté open source mondiale où la créativité humaine collective garantit que la technologie serve tout le monde.

— Pas pour le profit, mais pour tous.

Façons de s'impliquer

OpenNHP est construit par la communauté. Il existe de nombreuses façons de contribuer.

💻

Contributions de code

Soumettez des pull requests pour des corrections de bugs, de nouvelles fonctionnalités ou des améliorations de performance. Consultez notre guide de contribution pour commencer.

Voir les pull requests
🐛

Signaler des problèmes

Bug découvert ? Demande de fonctionnalité ? Ouvrez une issue sur GitHub avec des informations détaillées pour nous aider à nous améliorer.

Ouvrir une issue
📚

Documentation

Aidez à améliorer notre documentation, écrivez des tutoriels ou traduisez le contenu dans d'autres langues.

Voir la documentation

Recherche en sécurité

Règles

Code de conduite

Nous nous engageons à offrir un environnement accueillant et inclusif à tous. Toutes les personnes participant à la communauté OpenNHP sont tenues de respecter notre Code de conduite.

  • ✓ Faire preuve de respect et d'inclusivité
  • ✓ Accueillir les nouveaux venus et les aider à apprendre
  • ✓ Se concentrer sur des retours constructifs
  • ✓ Présumer de bonnes intentions
